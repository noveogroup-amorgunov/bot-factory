## constructor/server

Set `VK_APP_TOKEN` variable in `.env` file.

```shell
$ npm install
$ node constructor/server/index.js
```

Open in url:

```url
http://localhost:8081/constructor
http://localhost:8081/constructor/pizza-bot
```
